Project Title
Event Management System(EventHub) (JSP & Servlets)
Project Description
This project is a Java-based web application developed using JSP, Servlets, JDBC, HTML, CSS, and MySQL.
It provides functionalities for managing users, events, enquiries, products, reviews, and client interactions through a dynamic web interface.
The application follows a MVC-style architecture, where JSP handles the view layer and Java classes handle business logic and database operations.

Features
User authentication (Login / Forgot Password)
Event listing and management
User & enquiry management
Product handling and cart functionality
Reviews and ratings system
Client view pages
Dynamic JSP pages with reusable headers & footers
PDF generation support
Image gallery and map integration

Technologies Used
Frontend: HTML, CSS, JSP
Backend: Java, Servlets, JDBC
Database: MySQL
Server: Apache Tomcat
Tools: Eclipse
